# 🪒 Navalha Barbearia - Sistema de Agendamento

Sistema completo para gestão de barbearia com agendamentos online.

## ✨ Funcionalidades

- **Área do Cliente**: Agendamento de horários, histórico de serviços
- **Área do Barbeiro**: Visualização semanal/diária, gestão de atendimentos
- **Painel Admin**: Gestão de serviços, barbeiros, relatórios financeiros
- **Múltiplos Pagamentos**: Pix, Cartão, Dinheiro, Boleto

## 🚀 Como usar

1. Abra o `index.html` no navegador
2. Use as credenciais demo:
   - Cliente: `cliente@navalha.com` / `1234`
   - Barbeiro: `barbeiro@navalha.com` / `1234`
   - Admin: `admin@navalha.com` / `1234` (em `admin.html`)

## 🛠️ Tecnologias

- HTML5, CSS3, JavaScript (ES6+)
- LocalStorage para persistência
- Design Glassmorphism
- Font Awesome Icons
